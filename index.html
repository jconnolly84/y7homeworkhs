Hardware 1: Input devices

Input devices send data into a computer so it can be processed.

Key idea

A computer needs information to work with. Input devices capture data (like text, sound, images, movement) and convert it into a form the computer can understand.

Common examples

  • Keyboard ‚Äì text and commands
  • Mouse / trackpad ‚Äì pointer movement and clicks
  • Microphone ‚Äì sound input
  • Webcam ‚Äì images and video
  • Scanner ‚Äì paper documents into digital images
  • Touchscreen ‚Äì taps and swipes
  • Sensors ‚Äì temperature, light, motion (often used in phones and smart devices)

Hardware 2: Output devices

Output devices present results from a computer to the user.

Key idea

Once a computer has processed data, it produces useful information. Output devices make that information understandable to people.

Common examples

  • Monitor / screen ‚Äì visual output (text, images, video)
  • Speakers / headphones ‚Äì sound output
  • Printer ‚Äì paper copies of documents (hard copy)
  • Projector ‚Äì large display for a room

Hardware 3: Storage

Storage keeps data safe so it can be used later.

Primary vs secondary storage

Primary storage (like RAM) is used while the computer is running. It is very fast but temporary ‚Äî when the power is off, the data is lost.

Secondary storage (like an SSD or hard drive) keeps data permanently. It is usually slower than RAM but stores files even when the power is off.

Common storage devices

  • SSD ‚Äì fast, silent, no moving parts
  • HDD ‚Äì larger and often cheaper, but has moving parts
  • USB flash drive ‚Äì portable, convenient
  • Memory card ‚Äì used in phones, cameras, consoles
  • Cloud storage ‚Äì stored on the internet (access anywhere with login)

Software types (KS3)

Hardware is the physical part of a computer. Software is the programs that run on it. At KS3, we usually split software into three useful groups:

Type What it does Examples
Application software Helps the user do a task. Word processor, browser, game, photo editor
System software Runs the computer and supports applications. Operating system, device drivers
Utility software Maintains, protects or improves the system. Antivirus, backup, compression, cleanup tools

Quick check

Q: Is Microsoft Word system software or application software? Why?

What does an operating system do?

The operating system (OS) is the most important system software. Without it, most computers would be unusable. The OS:

  • Provides a user interface (windows, icons, menus)
  • Manages memory (RAM) so programs can run
  • Shares CPU time between programs (multitasking)
  • Manages files and folders (saving, opening, permissions)
  • Controls devices like keyboards, printers and speakers (often using drivers)
  • Helps with security (accounts, passwords, updates)

Utility software

Utility software helps keep a computer working well and keeps data safe. Utilities are usually not used to create work (like a word processor) ‚Äî they look after the system.

  • Antivirus ‚Äì detects and removes malware
  • Backup ‚Äì makes copies of files in case something goes wrong
  • Compression ‚Äì makes files smaller (e.g. zip files)
  • Disk cleanup ‚Äì removes temporary files to free space
  • Firewall ‚Äì helps block unauthorised access

Conclusion: How hardware and software work together

A computer needs both hardware and software to be useful.

The big picture

Think of hardware as the body of a computer and software as the instructions it follows. Hardware without software is like a phone with no apps and no operating system ‚Äî it can‚Äôt do much. Software without hardware can‚Äôt run.

Input ‚Üí Process ‚Üí Output (+ Storage)

Most computer systems follow the same pattern:

  1. Input ‚Äì data enters the system (keyboard, microphone, sensors)
  2. Process ‚Äì the CPU and memory work on it (software tells it what to do)
  3. Output ‚Äì information is shown to the user (screen, speakers, printer)
  4. Storage ‚Äì data can be saved for later (SSD, HDD, cloud)
